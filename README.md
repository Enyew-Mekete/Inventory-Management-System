# Inventory Management System

## 📌 Overview
The **Inventory Management System** is a Python-based application designed to help businesses efficiently track, manage, and update their stock levels. It provides a simple yet powerful interface for adding, updating, and monitoring inventory items, ensuring smooth operations and reducing errors in stock handling.

## ⚙️ Features
- **Add Items**: Insert new products with details such as name, category, quantity, and price.  
- **Update Stock**: Modify item quantities when products are sold or restocked.  
- **Search Items**: Quickly find products by ID, name, or category.  
- **Delete Items**: Remove outdated or discontinued products.  
- **View Reports**: Generate summaries of current stock, low inventory alerts, and sales history.  
- **Database Integration**: Uses SQLite for reliable and lightweight data storage.  

## 🛠️ Tech Stack
- **Python** (Core logic and backend)  
- **SQLite** (Database management)  
- **Flask** (Optional web interface)  
- **VS Code** (Development environment)
